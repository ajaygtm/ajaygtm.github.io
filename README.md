My Personal Portfolio Site!!!

## Local development

This Astro project requires Node.js 18.20.8 or newer.

If you use `nvm`, run:

```bash
nvm use
rm -rf node_modules package-lock.json
npm install
npm run dev
```

If `nvm use` says the version is not installed yet, run:

```bash
nvm install 22.20.0
nvm use 22.20.0
rm -rf node_modules package-lock.json
npm install
npm run dev
```
